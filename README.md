# AI業務体験シミュレーター

AIがあなたの業務をどう変えるか、今すぐ体験できるインタラクティブなデモサイトです。

## 🎯 機能

### 💰 コスト削減診断（全業種向け）
- 時給、人数、削減時間を入力するだけ
- 年間コスト削減額を自動計算
- 視覚的な棒グラフで効果を確認

### 🏗️ AI日報作成（建設業向け）
- 現場の簡単なメモを入力
- AIが構造化された日報に自動変換
- フォーマットを気にせず入力できる

### 🍝 AI POP生成（飲食業向け）
- メニュー情報とターゲット層を選択
- ターゲットに合わせたキャッチコピーを生成
- ビジュアルPOPデザインをプレビュー

## 📱 対応環境

- ✅ PC（Windows/Mac/Linux）
- ✅ タブレット
- ✅ スマートフォン
- ✅ モダンブラウザ（Chrome, Firefox, Safari, Edge）

## 🚀 使い方

1. ウェブサイトにアクセス
2. 3つのタブから体験したい業種を選択
3. 必要な情報を入力してボタンをクリック
4. 結果を確認

## 🛠️ 技術スタック

- **HTML5**: セマンティックなマークアップ
- **Tailwind CSS**: モダンなUIデザイン
- **Vanilla JavaScript**: シンプルで高速な動作
- **レスポンシブデザイン**: 全デバイス対応

## 📦 導入方法

### 方法1: GitHub Pagesで公開（推奨）

1. このリポジトリをフォーク
2. Settings → Pages → Branch: main を選択
3. Save をクリック
4. `https://ユーザー名.github.io/ai-simulator/` でアクセス可能

### 方法2: ローカルで実行

```bash
# ファイルをダウンロード
git clone https://github.com/あなたのユーザー名/ai-simulator.git
cd ai-simulator

# index.htmlをブラウザで開く
open index.html  # Mac
start index.html # Windows
```
